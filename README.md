# Jacob Miller — GitHub Curriculum

An interactive, static portfolio-style site showcasing my GitHub repositories across multiple domains: **Foundations, Full Stack, Security, Real Estate, Blockchain, and Capstone projects**. The site features modern design, smooth animations, hover effects, glassmorphism panels, and subtle interactive particles. Fully client-side — no backend required.

---

## Features

- Multi-section curriculum with dynamic repo grids.
- Interactive hover/tilt repo cards.
- Typing intro animation on load.
- Dark/light theme toggle.
- Glassmorphism panels with gradient backgrounds.
- Subtle cursor-following particle animations.
- Responsive design for desktop and mobile.
- Downloadable PDF version of the curriculum.
